# NetisOS
This is a Joke OS written in C++

Yasen huy, rakam ded ebet kobuly

## Requirements
* GCC compiler
* Qemu emulator
* Makefile

## Compilation
```
cd bootloader
make
cd bootloader
make
cd ../../kernel
make
make run
```
