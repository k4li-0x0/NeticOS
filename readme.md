# NeticOS
This is a joke OS written on C++
Rakam ded ebet kobuly, yasen huy

## ❗️ Requirements
- GCC compiler
- QEMU emulator
- Makefile

## ❓ How to run
Use
```
make all
```
or
```
cd kernel && make
cd bootloader && make
cd kernel && make run
```
